!!! info "补充"
    个人假人可以是自己的名字。  
    假人为公共资源，所有玩家均可打开假人内背包，无论什么名字。

!!! tip "规范"
    若可能出现多个同类假人，请在后面加数字以保持规范。例如`Slime0`-`Slime1`-`Slime2`，建议从 0 开始。  
    若有必要使用临时假人，请命名为`tmp`/`tmp0`。
